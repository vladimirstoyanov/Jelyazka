About Jelyazka

Jelyazka is a RSS reader program. If you don't know what RSS is, check these links:
  • http://www.whatisrss.com/
  • http://en.wikipedia.org/wiki/RSS

Main window

The above image is showing a main window of Jelyazka program. With a combobox widget you can change the RSS feeds. With buttons and you can browse the articles of current RSS feed (current RSS feed of the above image is "The Verge - All Posts").

The above image is showing an article. On the top with bold font is a title, bottom the title is a link to source article, bottom the link is a description.

If you want to add a RSS feed(s), you can do that, when click on button. To change settings use this button. To refresh current feed, use this button.

Add rss feed(s) window

In this window you can add new RSS feed(s), when you type an URL address in this "edit box" and after that, click on button. If you typed the right address of a RSS feed, the program add it momently to the bottom table. If an URL isn't the right address of a RSS feed, then when click on , it change the name of button to and begin searching of RSS feeds in this web page. When found RSS feed, the program add it in the table and continue searching of RSS feeds while you click button, close the window or searching of RSS feeds finish.

RSS feeds from the table can be added in the program database, when you click on button. If you don't want any RSS feed you can uncheck it or you can remove it when select a row and click button, then this row will be removed from the table.

Options window

Above image shown Options window. At left part of the window you can see four options: Collection feeds, Filters, Notifications and Proxy connection.

Content

Collection feeds

In this option you can see two lists. The left list contain collection of RSS feeds. Here are every RSS feeds that you was added (or default added). The right list contain RSS feeds, who will be appear on the main window. If you want to add RSS feed(s) from left list to the right, you must select RSS feeds from the left list and click on button. If you want to delete RSS feed(s) from the right list you must select RSS feed(s) from it and click on button. You can use search box to find RSS feed from the left list. If you want to remove RSS feed(s) from the left list, you must select RSS feed(s) from it and click button.

Content

Filters

If you don't want to read all data from RSS feed(s), you can add some filters, then on the main window will appear only articles who contains some string from the "Filter List".

Content

Notifications

With notification options you can change the time (1-60 minutes) of reload RSS feed(s) and enable/disable "Notification window" . "Notification window" appear, when the program found new articles of current RSS feeds.

Content

Proxy connection

The program can use proxy server for its connections. If you don't know what proxy server is, check this link:
http://en.wikipedia.org/wiki/Proxy_server

With "Enable proxy server connection" check box, you can enable/disable using of proxy server from the program.
