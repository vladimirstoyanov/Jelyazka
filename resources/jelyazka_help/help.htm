About Jelyazka

Jelyazka is an RSS reader program. If you don't know what RSS is, check these links:
  • http://www.whatisrss.com/
  • http://en.wikipedia.org/wiki/RSS

Main window

The above image shows the main window of the program. With this combobox you can choose what RSS feed to read. With these buttons and you can browse articles of choosen RSS feed (for example: the current RSS feed of the above image is "The Verge - All Posts").

The above image shows an article. A title of the article is situated on the top, formatted with bold font. Bewol the title is situated a link to the source article. Bewol the link is situated a description of the article.

If you want to add an RSS feed or feeds, you can do that, when click on button. To change settings use this button. To refresh the current feed, use this button.

Add RSS feed(s) window

In this window you can add a new RSS feed or feeds. You can do it, when you type an URL address in this "edit box" and after that, click on button. If you have typed the right address of RSS feed, the program add this one to the below table. If the URL address isn't the right one, then when click on , it changes the name of button to and begins to search RSS feeds in this web page. When finds RSS feed, the program add it to the table and continue searching of RSS feeds while you click button, close the window or searching of RSS feeds finish.

The RSS feeds from the table can be added in the program database, when you click on button. If you don't want any RSS feed you can uncheck it or you can remove it when select the row of this feed and click button.

Options window

The above image shows 'Options' window. At the left part of the window you can see four options: Collection feeds, Filters, Notifications and Proxy connection.

Content

Collection feeds

In this option you can see two lists. The left list contain collection of RSS feeds. There are the RSS feeds that you have added. The right list contain RSS feeds which appear on the main window. If you want to add some RSS feed(s) from the left to the right list, you must select RSS feeds from the left one and then click on button. If you want to delete RSS feed(s) from the right list you must select RSS feed(s) from this list and click on button. You can use search box to find RSS feed(s) from the left list. If you want to remove RSS feed(s) from the left list, you must select RSS feed(s) and click button.

Content

Filters

If you don't want to read all data from RSS feed(s), you can add some filters, then on the main window will appear only articles which contains some string from the "Filter List".

Content

Notifications

With notification options you can change reload time (1-60 minutes) of RSS feed(s) and enable/disable "Notification window" It appear, when the program found new articles.
